# Koopa

Library for generating/parsing/optimizing Koopa IR.

## Building from Source

Make sure the Rust toolchain is installed on your computer, and then run:

```sh
git clone --recursive https://github.com/pku-minic/koopa.git
cd koopa
cargo build --release
```
